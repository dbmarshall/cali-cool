## Installation

1. Clone repo and install NPM packages:

    ```
    git clone https://github.com/dbmarshall/cali-cool
    cd cali-cool/
    yarn install 
    cd client/
    yarn install 
    cd ../
    ```

2. Start server (local environment only):

    ```
    yarn start
    ```
